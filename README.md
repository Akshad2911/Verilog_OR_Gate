# Verilog_OR_Gate Using Verilog HDL

## Description

This repository contains the Verilog HDL implementation of a 2-input OR gate along with its testbench for functional verification. The project demonstrates the design and simulation of a basic combinational logic circuit using Verilog HDL. The functionality is verified by applying all possible input combinations and observing the corresponding output.

### Features

* 2-input OR gate implementation in Verilog HDL
* Simple and modular design
* Comprehensive testbench for verification
* Simulation of all input combinations
* Compatible with Vivado, ModelSim, and Icarus Verilog

### Files

* `or_gate.v` – Verilog module implementing the OR gate.
* `README.md` – Project documentation.

### Applications

* Digital logic design
* FPGA and ASIC design learning
* Verilog HDL practice
* Digital electronics laboratory experiments
* Building block for combinational logic circuits
